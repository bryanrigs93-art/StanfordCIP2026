# Week 1 — Hospital Karel 🏥

## Overview

This exercise introduced me to **Karel the Robot** and the idea of solving problems step by step.

Karel moves across the world, checks for beepers, and builds a small hospital whenever one is found.

## Main Concepts

* Functions
* Decomposition
* Loops
* If statements
* Debugging logic

## What I Practiced

I practiced breaking one big task into smaller helper functions. Instead of writing all the steps in one place, I organized the solution so each function had a clear job.

## Takeaway

This problem helped me understand that programming is easier when I divide a problem into small, reusable steps.
