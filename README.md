## Week 7 / 8 — Final Project

My final project is **My Perfect Gift Chatbot**, a personalized gift recommendation assistant.

The project connects Python foundations with a real business idea. It uses a CSV catalog of products, asks the customer a few questions, filters the best options with rule-based logic, and calls GPT to generate a clearer recommendation.

The goal is to show the connection between business and technology: a small business can use structured data, automation, and AI-assisted workflows to improve the customer experience.

**Key concepts:**

* Chatbot workflow
* CSV catalog data
* Lists and dictionaries
* Conditional logic
* Product scoring
* GPT-assisted recommendations
* Text layout
* User experience thinking
* Business idea presentation
* Tech-business connection

## Extra Content — Coding With AI

As part of the extra Code in Place content, I explored how AI can support the programming process.

One of the most interesting topics for me was learning how **AI coding agents** can help developers build projects faster when they are given clear instructions, context, and feedback.

This helped me understand that AI coding tools are not just for generating code. They can also support the full development workflow:

* Understanding the goal of an app
* Asking questions about requirements
* Suggesting a technical approach
* Creating files and project structure
* Explaining how to run the project
* Helping debug errors
* Improving code after testing
* Producing setup instructions and documentation

A key lesson I learned is that the quality of the result depends on how clearly I explain the problem. When I describe what I want, what is not working, and what the expected behavior should be, the AI tool can give better guidance.

This connects strongly with programming foundations. Before this course, I mostly experimented with AI tools through trial and error. Now, I can understand code better, ask more specific questions, identify bugs more clearly, and guide AI tools with more confidence.

For me, coding with AI is not about replacing learning. It is about using AI as a support tool while still understanding the logic behind the program.

## Extra Project — Ice Cream Sales Forecast

In the **Extra Content — Coding With AI** section, I also worked on a beginner-friendly data project called **Ice Cream Sales Forecast**.

The project uses a CSV file with ice cream sales data.

The goal is to read historical data, analyze the relationship between temperature and sales, and create a simple prediction.

**Key concepts:**

* CSV reading
* Lists
* Dictionaries
* Loops
* Functions
* Basic forecasting logic
* AI-assisted coding practice
* Beginner-friendly data analysis

This project helped me understand that machine learning and forecasting start with data. Before making predictions, Python needs to read, organize, and process information.
