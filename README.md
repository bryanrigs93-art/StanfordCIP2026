# Stanford Code In Place 2026 – Python Practice

This repository is part of my learning journey in **Stanford Online's Code In Place 2026**.

Here I am organizing my Python practice, Karel exercises, small projects, and notes while I continue learning programming step by step.

## About This Repository

My goal with this repo is to keep track of what I learned and show how I started building stronger programming foundations.

Some of the main topics I practiced were:

- Karel basics
- Python foundations
- Functions
- Loops
- Conditional logic
- Graphics
- Lists and dictionaries
- CSV files
- Small project ideas
- Coding with AI

## Karel Foundations

One of the first things we learned was how to control Karel using basic Python commands:

```python
move()
turn_left()
put_beeper()
pick_beeper()
```

I also practiced ideas like:

```python
while front_is_clear():
```

This helped me understand how programs follow instructions, check conditions, and solve problems one step at a time.

## Final Project

For my final project, I worked on **My Perfect Gift Chatbot**, a small idea that connects Python foundations with a real business workflow.

The project uses simple logic, product data, and AI-assisted recommendations to help users choose gift options.

## Main Takeaway

Code In Place helped me understand that programming is not just about memorizing syntax.

It is about learning how to think logically, break problems into smaller parts, and use code to build creative and practical solutions.
