# Week 4 — Python Control Flow

## Overview

This exercise helped me practice how Python makes decisions and repeats actions.

The program is a simple High-Low game where the player sees their number and guesses if it is higher or lower than the computer's hidden number.

## Main Concepts

- Random numbers
- Constants
- For loops
- While loops
- If / else statements
- Boolean logic
- Input validation
- Score tracking

## File

- `High_Low_Game.py` — High-Low guessing game
