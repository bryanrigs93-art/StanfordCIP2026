# Week 2 — Art of Karel 🎨🤖

## Overview

This exercise helped me practice more advanced Karel problem solving.

In this challenge, Karel collects beepers from one place and spreads them across the row using clear movement logic and helper functions.

## Main Concepts

* Stepwise refinement
* Helper functions
* Karel movement logic
* Decomposition
* Clean program structure

## What I Practiced

I practiced planning the solution before writing the code. The main goal was to divide the problem into smaller actions so Karel could move, collect, and place beepers in an organized way.
