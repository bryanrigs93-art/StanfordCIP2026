## Karel Movement Song

This song was created just for fun while putting into practice the first basic Karel commands we learned in the first Python lesson:

```python
move()
turn_left()
put_beeper()
pick_beeper()
```

It also connects with:

```python
while front_is_clear():
```

The goal was to make these programming concepts easier to remember through music.
