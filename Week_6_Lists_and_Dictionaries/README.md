# Week 6 — Lists and Dictionaries

## Overview

This exercise introduced how Python can store and organize multiple pieces of data.

The Heads Up program reads words from a text file, stores them in a list, and selects a random word.

## Main Concepts

- Lists
- Strings
- File reading
- Loops
- Helper functions
- Random choice
- Basic data organization

## Files

- `Heads_Up.py` — Reads words from a file and selects a random word.
- `cswords.txt` — Word list used by the program.
