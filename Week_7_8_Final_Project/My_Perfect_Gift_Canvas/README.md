## Overview

This is the visual version of my final project idea: My Perfect Gift, a personalized gift recommendation assistant.

The focus of this project is not only the visual layout. The main idea is to show how Python can connect business needs with technology solutions.

The project represents a simple business workflow where a customer needs help choosing a personalized gift. The interface presents the idea in a clear and organized way, similar to how a small business could explain or prototype a digital assistant.

## Project Essence

My Perfect Gift connects three important areas:

- Business idea presentation
- User experience thinking
- AI-assisted recommendation logic

The console version works as a chatbot connected to GPT, while this version helps present the business concept visually.

Together, both versions show the same idea from two angles:

1. A chatbot experience for the customer.
2. A visual layout to communicate the product and business workflow.

## Technology and Business Connection

This project helped me understand how programming can support a real business concept.

Instead of only writing isolated code, I used Python to think about:

- How a customer interacts with a recommendation assistant.
- How a business catalog can be used as structured data.
- How GPT can help generate a clearer customer-facing recommendation.
- How text layout can guide the user through the experience.
- How a simple prototype can communicate a business idea.
