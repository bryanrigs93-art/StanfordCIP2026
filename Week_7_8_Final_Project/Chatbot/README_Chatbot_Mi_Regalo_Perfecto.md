## Overview

This folder contains the final project chatbot experience for **My Perfect Gift**, a personalized gift recommendation assistant.

The project connects a small business idea with Python, CSV data, GPT-assisted recommendations, and user experience thinking. The goal is to show how technology can support a real customer workflow: helping someone choose a personalized gift from a catalog.

## How It Works

1. The user chooses English or Spanish.
2. The chatbot explains the gift recommendation flow.
3. Python loads the product catalog from a CSV file.
4. The program filters products using rule-based logic.
5. GPT is called to generate a clearer customer-facing recommendation.
6. The user can return to the main menu or finish the interaction.

## CSV Catalog

The project uses this CSV file as the product catalog:

```text
catalogo_real_mi_regalo_perfecto.csv
```

The CSV contains real product options, prices, delivery estimates, personalization options, occasions, and notes for the chatbot.
