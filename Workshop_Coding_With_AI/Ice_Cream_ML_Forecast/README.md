# Ice Cream Sales Forecast

This extra project explores a beginner-friendly machine learning workflow using Python.

It uses a small CSV dataset to practice loading data, choosing features, training a Linear Regression model, checking simple metrics, and making a basic prediction.

This is a learning demo only, not a real business forecasting model.
