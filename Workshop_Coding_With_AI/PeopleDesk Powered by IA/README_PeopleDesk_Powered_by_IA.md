# PeopleDesk Powered by IA

Learning project created for Stanford Code in Place.

This project is a Python simulation that explores how a manager-style assistant can organize CSV data, generate simple reports, and support review workflows.

Project link: https://codeinplace.stanford.edu/cip6/share/zNop9IMl04UcxtGAy1il

## Main idea

The app uses Python, CSV data, a terminal menu, canvas graphics, and optional GPT assistance to create a guided business-style workflow.

## How to run

```bash
python main.py
```

Requires `employees_smart_company.csv` in the same folder.

## Files

- `main.py` — Main Python program.
- `employees_smart_company.csv` — Sample dataset.
- `README_PeopleDesk_Powered_by_IA.md` — Project documentation.
