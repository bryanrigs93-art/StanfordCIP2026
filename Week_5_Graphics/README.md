# Week 5 — Graphics

## Overview

This exercise introduced the basics of drawing with Python Graphics.

The program creates a canvas and draws random circles in different positions and colors.

## Main Concepts

- Graphics Canvas
- Constants
- For loops
- Random numbers
- Helper functions
- Coordinates
- Drawing ovals

## File

- `Random_Circles.py` — Random circles graphics program
